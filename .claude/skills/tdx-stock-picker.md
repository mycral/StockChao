# 通达信选股公式编写技能

## 技能用途

当你被要求写通达信**选股公式**时，加载此技能并按照本指南执行。

## 核心原则

1. 选股公式**最后一行必须是条件表达式**，返回 1 表示满足条件应被选出
2. 选股公式中**不应使用未来函数**（ZIG、PEAK、TROUGH、REFX等），否则信号会漂移
3. 选股公式中**不应有输出语句**（`:` 而非 `:=`），只用赋值和条件判断
4. 优先使用**当日可确认的数据**，避免使用当日盘中不确定的数据
5. 结合**基本面条件**时，注意使用 FINANCE、DYNAINFO 等函数

## 选股公式结构

```
第一部分：技术指标条件定义
A1:= 条件表达式1;
A2:= 条件表达式2;
A3:= 条件表达式3;

第二部分：过滤条件（可选）
过滤:= 基本面过滤 AND 市场过滤 AND 停牌过滤;

第三部分：最终输出（只返回 0 或 1）
A1 AND A2 AND A3 AND 过滤;
```

## 常用选股条件模式

### 1. 均线条件

```tdx
{ 均线多头排列：5>10>20 且持续5天以上 }
MA5:=MA(CLOSE,5);
MA10:=MA(CLOSE,10);
MA20:=MA(CLOSE,20);
多头:=MA5>MA10 AND MA10>MA20;
COUNT(多头,5)=5;
```

### 2. 量价条件

```tdx
{ 成交量放大：今日量是20日均量的2倍，且股价上涨 }
VOL>MA(VOL,20)*2 AND CLOSE>REF(CLOSE,1);
```

### 3. 突破条件

```tdx
{ 股价突破20日最高价 }
CLOSE>=HHV(HIGH,20) AND CLOSE>OPEN;
```

### 4. 缩量回踩

```tdx
{ 缩量回踩10日均线 }
MA10:=MA(CLOSE,10);
回踩:=ABS(L-MA10)/MA10<0.02;
缩量:=VOL<MA(VOL,20)*0.5;
回踩 AND 缩量 AND CLOSE>MA10;
```

### 5. 均线金叉

```tdx
{ 5日均线上穿10日均线，且放量 }
金叉:=CROSS(MA(CLOSE,5),MA(CLOSE,10));
放量:=VOL>MA(VOL,20)*1.5;
金叉 AND 放量;
```

### 6. 涨停选股

```tdx
{ 普通票涨停 }
涨停:=CLOSE>=REF(CLOSE,1)*1.0995 AND HIGH=CLOSE;
涨停;
```

```tdx
{ 今日首次涨停 }
涨停:=CLOSE>=REF(CLOSE,1)*1.0995 AND HIGH=CLOSE;
首停:=涨停 AND COUNT(涨停,20)=1;
首停;
```

### 7. 跌反弹

```tdx
{ 连续3天下跌后缩量小阳线 }
连跌:=DOWNNDAY(CLOSE,3);
小阳:=CLOSE>OPEN AND (CLOSE-OPEN)/OPEN<0.02;
缩量:=VOL<MA(VOL,20)*0.7;
连跌 AND 小阳 AND 缩量;
```

## 基本面筛选

### 市盈率筛选

```tdx
{ 市盈率在合理区间 }
DYNAINFO(40)>0 AND DYNAINFO(40)<50;
```

### 换手率筛选

```tdx
{ 换手率 > 5% }
DYNAINFO(37)>5;
```

### 流通市值筛选

```tdx
{ 流通市值 < 100亿（单位：元） }
FINANCE(7)<10000000000;
```

### 财务指标筛选

```tdx
{ 净利润正增长 }
净利增长:=FINANCE(30)>REF(FINANCE(30),1);
净资产收益率:=FINANCE(30)/FINANCE(19)>0.1;
净利增长 AND 净资产收益率;
```

### 巴菲特选股法则

```tdx
销售毛利率:=(FINANCE(20)-FINANCE(21))/FINANCE(20)*100;
净利率:=FINANCE(30)/FINANCE(20)*100;
净资产收益率:=FINANCE(30)/FINANCE(19)*100;
销售毛利率>30 AND 净利率>15 AND 净资产收益率>15;
```

## 市场与板块过滤

```tdx
{ 排除ST股票 }
非ST:=NOT(NAMELIKE('ST') OR NAMELIKE('*ST') OR NAMELIKE('SST'));
{ 排除停牌 }
非停牌:=VOL>0 OR DYNAINFO(8)>0;
{ 排除科创板（可根据需要调整） }
非科创:=NOT(CODELIKE('688'));
{ 沪深主板 }
主板:=INBLOCK('上证A股') OR INBLOCK('深证A股');
主板 AND 非ST AND 非停牌;
```

## 组合模板

```tdx
{
技术面: 均线多头 + 放量 + 突破
基本面: 净利润正增长 + 市盈率合理
市场面: 非ST + 非停牌 + 主板
}

{=== 技术条件 ===}
MA5:=MA(CLOSE,5);
MA10:=MA(CLOSE,10);
MA20:=MA(CLOSE,20);
多头:=MA5>MA10 AND MA10>MA20;
放量:=VOL>MA(VOL,20)*1.5;
突破:=CLOSE>MA20 AND CLOSE/HHV(HIGH,20)>0.95;
技术条件:=多头 AND 放量 AND 突破;

{=== 基本面 ===}
净利增长:=FINANCE(30)>REF(FINANCE(30),1);
估值合理:=DYNAINFO(40)>0 AND DYNAINFO(40)<80;
基本面:=净利增长 AND 估值合理;

{=== 市场过滤 ===}
非ST:=NOT(NAMELIKE('ST') OR NAMELIKE('*ST'));
非停牌:=VOL>0;
主板:=INBLOCK('上证A股') OR INBLOCK('深证A股');

{=== 最终输出 ===}
技术条件 AND 基本面 AND 非ST AND 非停牌 AND 主板;
```

## 重要注意事项

### 选股公式 vs 指标公式

| 区别 | 选股公式 | 指标公式 |
|------|---------|---------|
| 输出 | 不能有输出（`:}`） | 可以有输出（`:}`） |
| 返回值 | 条件为1选出 | 多为数值序列 |
| 未来函数 | 禁用 | 可用但有风险 |
| 绘图函数 | 禁用 | 可用 |

### 性能优化

- 避免在选股公式中使用复杂循环
- 尽量用 `COUNT`、`EVERY` 代替 `BARSLAST` 循环
- 避免使用 `CALCSTOCKINDEX` 跨品种引用（大范围选股时极慢）

### 常见错误

```tdx
{ ❌ 错误：使用了输出语句 }
MA5:MA(CLOSE,5);  { 不应输出，只赋值 }
选股:CLOSE>MA5;   { 选股公式不应有"选股:"这个输出名 }

{ ✅ 正确：全部使用赋值 }
MA5:=MA(CLOSE,5);
CLOSE>MA5;
```

```tdx
{ ❌ 错误：使用了未来函数 }
金叉:=CROSS(MA(CLOSE,5),MA(CLOSE,10));
未来信号:=REFX(金叉,1);  { REFX是未来函数，选股禁用 }

{ ✅ 正确：只用当期数据 }
金叉:=CROSS(MA(CLOSE,5),MA(CLOSE,10));
金叉;
```

### 日期/时间注意事项

```tdx
{ DATE 返回值需要 +19000000 才等于实际日期 }
{ 例如返回 1000101 代表 2000年1月1日 }
DATE+19000000=20260301;  { 筛选2026年3月1日的数据 }

{ 更好的方式：用 REFDATE 引用特定日期 }
REFDATE(CLOSE,20260301);
```

## 快捷函数备忘

| 需求 | 函数 | 示例 |
|------|------|------|
| 排除ST | `NAMELIKE('ST')` | `NOT(NAMELIKE('ST'))` |
| 排除停牌 | `DYNAINFO(8)>0` 或 `VOL>0` | `VOL>0` |
| 代码筛选 | `CODELIKE('600')` | 上证主板 |
| 涨停价 | `ZTPRICE(REF(C,1),0.1)` | 10%涨停价 |
| 跌停价 | `DTPRICE(REF(C,1),0.1)` | 10%跌停价 |
| 创历史新高 | `CLOSE=HHV(HIGH,0)` | |
| 创历史新低 | `CLOSE=LLV(LOW,0)` | |

## 执行流程

当你需要写选股公式时：

1. **明确需求**：用户想要什么类型的股票？
   - 均线多头？突破？金叉？基本面？
2. **确定条件**：列出所有条件表达式
3. **检查过滤**：加入市场过滤（ST、停牌等）
4. **验证语法**：确保没有输出语句、没有未来函数
5. **优化性能**：简化计算、避免跨品种引用

## 示例：完整选股公式

**需求**：5日均线上穿10日均线，涨幅3%-8%，非ST，非停牌，市盈率<50

```tdx
{ 均线金叉 }
MA5:=MA(CLOSE,5);
MA10:=MA(CLOSE,10);
金叉:=CROSS(MA5,MA10);

{ 涨幅条件 }
涨幅:=(CLOSE-REF(CLOSE,1))/REF(CLOSE,1)*100;
合理涨幅:=涨幅>3 AND 涨幅<8;

{ 基本面 }
市盈率合理:=DYNAINFO(40)>0 AND DYNAINFO(40)<50;

{ 市场过滤 }
非ST:=NOT(NAMELIKE('ST') OR NAMELIKE('*ST'));
非停牌:=VOL>0;

{ 最终输出 }
金叉 AND 合理涨幅 AND 市盈率合理 AND 非ST AND 非停牌;
```
